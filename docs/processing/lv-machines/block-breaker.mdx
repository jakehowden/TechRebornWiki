**Block Breaker** is a Low-Voltage utility machine in Tech Reborn. It breaks the block in front of its output face and drops it as an item, automating block harvesting.

## Behavior

- Faces a single block position determined by the direction the machine is placed.
- Consumes 5 E/t (configurable) while actively breaking.
- Break time scales with block hardness: hardness-1 blocks take 100 ticks (5 s) by default.
- Drops the broken block's item into the world or into an adjacent inventory.
- Does not break unbreakable blocks (bedrock, barriers, etc.).
