# Advanced Machine Casing

**Advanced Machine Casing** is a structural block in Tech Reborn. It is used as wall blocks in several multiblock structures and as a crafting ingredient in MV-tier recipes.

## Recipe

<RecipeFromData id="techreborn:crafting/advanced_machine_casing" />

## Behavior

- Can be placed as a decorative casing block.
- Used as the casing block for multiblock structures including the Industrial Centrifuge and Industrial Grinder.
- Heat capacity: 1700 / 25 = 68 (used in multiblock heat calculations).
