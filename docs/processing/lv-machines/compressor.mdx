## Overview
The Compressor is an essential early-game LV machine. It compresses Ingots into Plates, which are used in many machine and tool recipes.

## Behavior
- **Input:** 1 Item Slot
- **Output:** 1 Item Slot
- **Power:** 32 E/t (from Low Voltage)
- **Special:** Can be sped up using Overclocker Upgrades.
