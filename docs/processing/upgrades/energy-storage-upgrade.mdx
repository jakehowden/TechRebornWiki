# Energy Storage Upgrade

**Energy Storage Upgrade** is a machine upgrade in Tech Reborn. Each installed copy adds 40,000 E (default) to the machine's internal buffer.

## Recipe

<RecipeFromData id="techreborn:crafting/energy_storage_upgrade" />

## Effect

Per upgrade installed (default, configurable in `techreborn.json`):

| Stat | Change |
|---|---|
| Internal buffer | +40,000 E |

## Notes

- Useful for machines that must process recipes requiring a large burst of energy.
- Does not affect input rate or processing speed.
- Stacks with multiple copies.
