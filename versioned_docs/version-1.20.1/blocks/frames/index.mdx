# Machine Frames

Machine frames are the primary structural ingredient for crafting Tech Reborn machines. Each tier of frame is required to build machines of the same tier.

- <ItemIcon id="techreborn:basic_machine_frame" size={24}/> [Basic Machine Frame](./basic-machine-frame.mdx) — used in LV machines
- <ItemIcon id="techreborn:advanced_machine_frame" size={24}/> [Advanced Machine Frame](./advanced-machine-frame.mdx) — used in MV machines
- <ItemIcon id="techreborn:industrial_machine_frame" size={24}/> [Industrial Machine Frame](./industrial-machine-frame.mdx) — used in HV machines and multiblocks
