# Player Detector

**Player Detector** is a utility machine in Tech Reborn. It monitors the surrounding area and emits a redstone signal when one or more players enter its detection range.

## Behavior

- Consumes 1 E/t while active (default).
- Maximum input: 32 E/t (Low Voltage).
- Maximum detection radius: 128 blocks (configurable in `techreborn.json`).
- Outputs a redstone signal (strength proportional to number of players detected).
- Right-clicking cycles between three detection modes: **All Players**, **Others** (not the block owner), and **You** (owner only).
