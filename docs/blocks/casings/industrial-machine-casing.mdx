# Industrial Machine Casing

**Industrial Machine Casing** is a structural block in Tech Reborn. It forms the walls of the highest-tier multiblock structures including the Industrial Blast Furnace, Vacuum Freezer, and Distillation Tower.

<RecipeFromData id="techreborn:crafting/industrial_machine_casing" />

## Behavior

- Can be placed as a decorative casing block.
- Required as wall/floor/ceiling blocks in all tier-3 multiblock structures.
- Heat capacity: 2380 / 25 = 95.2 (the highest heat capacity of any casing tier).
