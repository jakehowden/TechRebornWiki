# Machine Casings

Machine casings are crafting ingredients and multiblock wall blocks. Each tier of casing corresponds to a multiblock that uses it as structure blocks.

- <ItemIcon id="techreborn:basic_machine_casing" size={24}/> [Basic Machine Casing](./basic-machine-casing.mdx) — used in basic multiblock walls
- <ItemIcon id="techreborn:advanced_machine_casing" size={24}/> [Advanced Machine Casing](./advanced-machine-casing.mdx) — used in industrial structures
- <ItemIcon id="techreborn:industrial_machine_casing" size={24}/> [Industrial Machine Casing](./industrial-machine-casing.mdx) — used in tier-3 multiblocks
