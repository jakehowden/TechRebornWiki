**Overclocker Upgrade** is a machine upgrade in Tech Reborn. Each installed copy speeds up processing and increases power draw.

## Recipe

<RecipeFromData id="techreborn:crafting/overclocker_upgrade" />

## Effect

Per upgrade installed (default values, configurable in `techreborn.json`):

| Stat | Multiplier |
|---|---|
| Speed | +25% faster (× 0.75 ticks per operation) |
| Power draw | +75% more per tick (× 1.75) |

Effects from multiple Overclocker Upgrades stack multiplicatively.

## Notes

- Accepted by most powered machines (electric furnaces, grinders, etc.).
- Installing too many without sufficient power infrastructure can stall the machine.
- Pairs well with [Transformer Upgrade](./transformer-upgrade.mdx) to accept higher-voltage input after overclocking.
