## Overview
The Rolling Machine is a crafting table that requires energy to function. While historically an MV machine for crafting plates, in recent versions it behaves as an advanced Auto Crafting Table.

## Behavior
- **Input:** 3x3 Crafting grid for ingredients.
- **Output:** 1 Item Slot
- **Power:** 32 E/t (Low Voltage default)

## Recipes accepted
*In Tech Reborn 1.20.1, the Rolling Machine may not have exclusive machine recipes, often acting as a powered standard crafter or disabled based on the modpack.*
