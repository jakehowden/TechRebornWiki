# Basic Machine Casing

**Basic Machine Casing** is a structural block in Tech Reborn. It is used both as a decorative block and as a crafting ingredient in tier-1 machine recipes.

## Recipe

<RecipeFromData id="techreborn:crafting/basic_machine_casing" />

## Behavior

- Can be placed as a decorative casing block.
- Used as a crafting ingredient in many LV machine recipes.
- Heat capacity: 1020 / 25 = 40.8 (relevant to Vacuum Freezer and Industrial Blast Furnace multiblock calculations).
