**Storage Buffer** is a Tech Reborn item storage block. It functions as an upgradeable chest with a configurable number of item slots.

## Recipe

<RecipeFromData id="techreborn:crafting/storage_buffer" />

## Behavior

- Provides a GUI-accessible item storage inventory.
- Can be upgraded with Storage Buffer Upgrader items to increase slot capacity.
- Unlike vanilla chests, it does not double when placed adjacent to another.
- Connects to item pipes and other transfer systems via its faces.
