# Diesel Generator

<MachineStats power={32} buffer={10000} tier="Low" />

## Overview
The Diesel Generator burns advanced fluid fuels like Diesel, Nitro Diesel, or standard Fuel to produce Energy. It is an excellent upgrade for your base once you have set up a distillation tower.
