**Iron Alloy Furnace** is a non-electric machine in Tech Reborn. It accepts two input items and a solid fuel source, producing alloys that require combining two materials in a single smelt — similar to the electric [Alloy Smelter](./alloy-smelter.mdx) but without any power requirement.

## Behavior

- Powered by solid fuel (coal, wood, etc.) rather than electricity.
- Two input item slots; one output slot.
- Processes alloy recipes: bronze (copper + tin), invar (iron + nickel), and similar.
- Slower than the electric Alloy Smelter and not upgradeable.
