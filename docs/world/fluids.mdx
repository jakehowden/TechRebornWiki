# Fluids

Tech Reborn introduces numerous fluids. Most of these fluids are not placed in the world directly but are kept internally in machines, tanks, and fluid cells.

Below is a non-exhaustive list of notable fluids used in processing:

| Fluid | Source / Production | Common Use |
| :--- | :--- | :--- |
| **Berylium** | Electrolyzer | High tier crafting |
| **Biofuel** | Industrial Grinder | Generator Fuel |
| **Calcium** | Electrolyzer | Chemical Reactor |
| **Carbon** | Electrolyzer | Crafting |
| **Chlorine** | Electrolyzer | Chemical Reactor |
| **Compressed Air** | Compressor | Empty Cells |
| **Deuterium** | Electrolyzer | Fusion |
| **Diesel** | Distillation Tower | Diesel Generator Fuel |
| **Helium** | Centrifuge / Fusion | Coolant Cells |
| **Hydrogen** | Electrolyzer | Generator Fuel |
| **Lithium** | Electrolyzer | Battery crafting |
| **Mercury** | Grinder / Centrifuge | Advanced grinding |
| **Methane** | Centrifuge | Gas Turbine |
| **Nitrogen** | Centrifuge | Coolant |
| **Nitrocarbon** | Chemical Reactor | Diesel Fuel |
| **Oil** | World Gen / Extraction | Distillation |
| **Oxygen** | Electrolyzer | General crafting |
| **Potassium** | Electrolyzer | Processing |
| **Silicon** | Electrolyzer | Solar panels |
| **Sodium** | Electrolyzer | Chemical Reactor |
| **Sodium Persulfate** | Chemical Reactor | Grinder additive |
| **Sulfur** | Electrolyzer | Battery crafting |
| **Sulfuric Acid** | Chemical Reactor | Battery crafting |
| **Tritium** | Centrifuge | Fusion |
