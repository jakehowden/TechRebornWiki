# Welcome to the Tech Reborn Wiki

Tech Reborn is a massive tech mod for Minecraft that introduces complex machinery, energy systems, and high-tech tools. It is heavily inspired by classic mods like IndustrialCraft 2 and GregTech but updated for modern versions of Minecraft.

## Getting Started
To get started with Tech Reborn, you should focus on gathering **Rubber** and basic ores like Iron, Copper, and Tin. 

1. **Find a Rubber Tree**: Obtain a Treetap and collect Sap.
2. **Generate Energy**: Build a <ItemRef id="techreborn:solid_fuel_generator" />.
3. **Basic Processing**: Construct an Electric Furnace, <ItemRef id="techreborn:grinder" />, and Extractor to vastly improve your resource yields.

Use the sidebar to explore detailed information about all machines, multiblocks, and materials!
