# Rubber Tree

Rubber Trees spawn naturally in swamps, forests, and other biomes. They are the primary source of Rubber, an essential resource for insulating cables and crafting machine parts.

## Obtaining Sap
Rubber tree logs occasionally generate with a distinct sap spot on one side. 
Using a [Treetap](../tools-armor/tools/treetap) on this spot will extract **Sap**. 

## Processing
- **Sap → Rubber:** Processing Sap in an [Extractor](../processing/lv-machines/extractor) is the most efficient way to obtain Rubber (yields 3). Smelting Sap in a furnace yields only 1.
- **Rubber Wood:** The wood can be crafted into planks or processed in an [Industrial Sawmill](../multiblocks/industrial-sawmill) to maximize yield.
