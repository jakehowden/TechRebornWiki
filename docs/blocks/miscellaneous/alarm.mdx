# Alarm

**Alarm** is a utility block in Tech Reborn. When powered by a redstone signal it plays a looping sound at the block's position.

## Recipe

<RecipeFromData id="techreborn:crafting/alarm" />

## Behavior

- Requires a redstone signal to activate; plays a repeating alert sound while powered.
- Right-clicking opens a GUI to select from multiple alarm sounds.
- No energy consumption — purely redstone-driven.
- Useful for automated base alerts, factory warnings, or decorative ambience.
