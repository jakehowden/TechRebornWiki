# Rock Cutter

The **Rock Cutter** is a non-electric tool that functions like a saw for stone-based blocks. It cuts stone, marble, and similar blocks into slabs and stair variants without a crafting table.

Unlike the electric drill and chainsaw series, the Rock Cutter uses standard durability rather than an internal energy buffer.
